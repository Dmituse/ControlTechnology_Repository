# EasyPhoneBook

Данный проект — это электронный телефонный справочник, реализованный на C++ с использованием Qt и SQL.
Приложение позволяет хранить, добавлять и отображать контакты, состоящие из имени, фамилии и одного или нескольких телефонных номеров.

Описание тестового задания: Необходимо разработать настольное приложение «Электронный телефонный справочник» на языке C++ с использованием Qt 6 и базы данных SQLite. Приложение предназначено для хранения контактной информации, включающей имена, фамилии и номера телефонов, при этом одному человеку может соответствовать несколько телефонных номеров.Программа должна предоставлять возможность добавления новых записей через графический интерфейс и отображения данных в табличном виде. В приложении требуется реализовать два режима отображения: сортировку по людям, при которой один человек отображается в одной строке со всеми его номерами, и сортировку по телефонным номерам, при которой каждая строка соответствует отдельному номеру, а имена могут повторяться.
